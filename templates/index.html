Global Exclusion Checker / File Parser

The purpose of this tool is to identify keywords from a Global Exclusion file within an Outlook file.
Some key info will also be parsed from the file for better readability (Subject, ID, To, From)

| Export Exclusions.csv from client site:
Admin > Site Settings > Integration > DealCloud Email Integration > Global Exclusions > Export

| Outlook file to be provided by client

| Supported file types:

File File Type
Exclusions .csv
Outlok - Email .eml
Outlook - Event .ics
Graph Explorer .json
